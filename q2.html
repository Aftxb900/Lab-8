Tim Berners-Lee: The Inventor of the World Wide Web

Tim Berners-Lee invented the World Wide Web in 1989, which revolutionized the way we access and share information across the globe. His invention created the foundation for the internet as we know it today and transformed the digital age.

The Three Key Components of the World Wide Web

First Component: HTML (HyperText Markup Language)
HTML is a standardized system for structuring documents on the web using markup tags. It defines the layout of web pages, allowing for elements like headings, paragraphs, and links to be formatted properly.
Second Component: HTTP (HyperText Transfer Protocol)
HTTP is the protocol used for transferring data between web browsers and servers. It facilitates the communication that allows users to retrieve resources, such as web pages and images, from the internet.
Third Component: URL (Uniform Resource Locator)
A URL is the address used to locate a specific resource on the web. It provides a way for users to navigate to websites, images, documents, and other content available across the World Wide Web.
